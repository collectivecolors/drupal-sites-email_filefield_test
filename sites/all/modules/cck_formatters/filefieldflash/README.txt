ABOUT

This module provides an additional formatter for CCK filefield fields to display
a SWF file as a flash file, using the swfobject_api module (required).  It supports
SWF files and image files; if a SWF file is uploaded, it will be displayed at
its native size.  If an image file is uploaded, it will be displayed at its native
size.  Any other file will be displayed using the default Filefield formatter.

REQUIREMENTS

- Drupal 6.x
- filefield module
- swfobject_api module
